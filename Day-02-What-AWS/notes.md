# Day 2 - What is AWS?

## What is AWS?

AWS (Amazon Web Services) is a cloud computing platform provided by Amazon.

It offers hundreds of cloud services that help businesses build, deploy, and manage applications without owning physical infrastructure.

## Common AWS Services

### EC2 (Elastic Compute Cloud)

Provides virtual servers for running applications.

### S3 (Simple Storage Service)

Provides secure object storage for files such as images, videos, PDFs, and backups.

### IAM (Identity and Access Management)

Used to securely manage users, groups, roles, and permissions.

### VPC (Virtual Private Cloud)

Allows you to create your own private network inside AWS.

## Advantages of AWS

- On-demand resources
- Pay-as-you-go pricing
- High scalability
- Global infrastructure
- Secure services
