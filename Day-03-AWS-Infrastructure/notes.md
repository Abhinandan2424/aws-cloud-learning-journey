# AWS Global Infrastructure

## What is Region?

A Region is a physical geographic location where AWS has multiple data centers.

## What is Availability Zone?

Availability Zones are isolated data centers within a Region that provide high availability and fault tolerance.

## What is Edge Location?

Edge Locations deliver content closer to users, reducing latency and improving performance.

## Key Points

- High Availability
- Fault Tolerance
- Low Latency
- Scalable Infrastructure
